Python Quests – Debugging Unit

 Overview
This project is a collection of Python scripts completed as part of the Debugging Unit assignment. It is structured as a series of 30 quests grouped into 6 levels, each focusing on key programming concepts. The goal is to develop problem-solving, debugging, and coding skills through progressive challenges.

How to Run
Clone the repository
git clone https://github.com/Becca4375/the_debugging_unit-python-quests-.git
Navigate into the project
cd the_debugging_unit-python-quests-
cd quests
run the file you want ./quest_name.py
